## 块级元素和行内元素的定义
块级元素
- 总是在新行上开始；
- 高度、行高以及外边距和内边距都可控制；
- 宽度缺省是它的容器的100%，除非设定一个宽度；
- 它可以容纳内联元素和其他块元素

行内元素
- 和其他元素都在一行上
- 高、行高及外边距和内边距都不可改变
- 宽度就是它的文字或图片的宽度，不可改变
- 内联元素只能容纳文本或其他内联元素

对行内元素需要注意的是：
- 设置宽度 width 无效
- 设置高度 height 无效，可以通过 line-height 来设置
- 设置 margin 只有左右 margin 有效，上下无效
- 设置 padding 只有左右 padding 有效，上下无效
